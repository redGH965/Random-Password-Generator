# Random Password Generator

**Автор:** [Ваше Имя Фамилия]
**Описание:** Простое GUI-приложение для генерации случайных паролей с настройкой параметров и сохранением истории.

## Возможности
- Настройка длины пароля (от 4 до 32 символов).
- Выбор типов символов: цифры, буквы, спецсимволы.
- Генерация пароля по клику.
- Сохранение истории в JSON.
- Отображение истории в таблице.

## Запуск
1. Установите Python 3.x.
2. Установите зависимости (если нужны): `pip install -r requirements.txt` (в данном проекте не требуется).
3. Запустите: `python main.py`
